# iAPSAdvisor

A SwiftUI-based iOS app targeting iOS 15+.

## Development

1. Open the project in Xcode.
2. Build and run on an iOS 15+ simulator or device.

## Features

- Displays a welcome message
- Allows you to pick `freeaps_settings.json` from the Files app and previews its contents
- Fetches Nightscout glucose entries, insulin events, and carb events for quick inspection

## Note

This project currently contains placeholder UI and no external dependencies.
